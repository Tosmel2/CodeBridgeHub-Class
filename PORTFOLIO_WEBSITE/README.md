# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML5** and **CSS3**. This portfolio showcases professional information, technical skills, featured projects, and contact details in a clean, elegant, and mobile-friendly design.

---

## 📸 Preview

This portfolio includes the following sections:

- 🏠 Hero Section
- 👨‍💻 About Me
- 💡 Skills & Technologies
- 🚀 Featured Projects
- 📩 Contact Section
- 📄 Footer

---

## ✨ Features

- Fully Responsive Design
- Modern Dark Theme

- Smooth Hover Animations
- Responsive Grid Layout

