# Student Management System UI

A polished frontend interface for connecting to the Student Management System API.

## Run the UI

1. Open `Frontend/index.html` in your browser.
2. Ensure the backend server is running at `http://localhost:5000`.
3. Use the form to create or update students and the table to manage the roster.

## Features

- Add new students
- Edit existing student records
- Delete students
- Refresh list with a single click
- Responsive and modern UI design
